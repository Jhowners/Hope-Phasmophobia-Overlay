### 高度な調査のためのHUDツール（ヘッドアップディスプレイ）

**記憶力との戦いを止め、ゴーストとの戦いを始めましょう。**

Phasmophobiaは、認知的な持久力が試されるゲームです。レヴナントがハントを開始したとき、あなたの脳は「闘争か逃走か」のモードに切り替わります。その一瞬で、足音の速度を計算したり、スマッジスティックのクールダウンを思い出そうとしたりすることは、**認知的負荷**を増やし、躊躇やミスを引き起こす精神的な摩擦となります。

**Hope Phasmophobia Overlay**は、あなたの「第二の脳」として機能します。ゲームの上に直接表示し、データ処理を代わりにやってくれることで、サバイバルに100%集中することを可能にします。

---

## 🧠 「脳の状態」の科学
Alt+Tabで攻略Wikiに切り替えたり、他のウィンドウを見たりするたびに、あなたの脳は**コンテキストスイッチのペナルティ**になります。脳が再集中するまでに数ミリ秒かかります—最高難易度の中、そんなに浪費できる時間はありません。

HPOは以下の機能により、あなたの脳を**常に最適な状態**に保ちます：
* **前注意処理：** 証拠が特定の色で光ります（🔴 赤 = EMF 5、🔵 青 = 氷点下の温度）。文字を読むことなく、これらのパターンを瞬時に認識できます。
* **筋肉記憶入力：** 移動を止めたりすることなく、ホットキー（`Alt+1`から`Alt+7`）で証拠を切り替えます。

## ⚡ 主要機能

### 1. リアルタイム推理
推測ではなく、確信を。
* **フィルタリング：** ホットキーで証拠をマークするにつれて、ゴーストリストが瞬時にフィルタリングされます。
* **スマートロジック：** ペーサーで「通常速度」をマークすると、レヴナントが自動的に除外されます。「オーブ」をマークすると、ミミック（偽オーブとしてフラグ付けしない場合）が除外されます。

### 2. ゴーストペーサー（BPM）
* **リズム同調：** ゴーストの足音のリズムに合わせて `F10` をタップします。
* **即時認識：** アプリはBPMを計算し、あなたの証拠リストと照合し、潜在的な一致するゴースト（例：ハントゥ、モーロイ、セーイ）をリアルタイムでハイライトします。

### 3. スマートスマッジタイマー
* **視聴覚キュー：** スピリット（180秒）、デーモン（60秒）、および通常ゴースト（90秒）のクールダウンを追跡します。
* **パニック管理：** 隠れ場所から安全に出られる正確なタイミングを知ることができます。

### 4. 3つの戦術モード
* **📖 分析モード：** トラックまたは安全な場所で使う、完全なジャーナルビュー。
* **🧠 インテルモード [HOME]：** ルーピング用に設計された、コンパクトなHUD。視界を遮ることなく重要な統計情報のみを表示します。
* **⏱️ サバイバルモード [PAUSE BREAK]：** UIをタイマーのみに絞り込みます。追跡の最中でも気を散らすものがゼロになります。

## 🎮 操作と使用法

**重要：** オーバーレイを表示するには、Phasmophobiaを**ボーダレスウィンドウ**モードで実行する必要があります。

### ホットキー
| キー | 機能 |
| :--- | :--- |
| **F7** | 十字架タイマー開始 / 再開 (0:25) |
| **F8** | スマッジタイマー停止 / リセット |
| **F9** | スマッジタイマー開始 / 再開 (3:00) |
| **F10** | ゴーストの速度（BPM）をタップで測定 |
| **F11** | 速度 / ペーサーをリセット |
| **Alt + 1-8** | 証拠の切り替え (EMF、ボックス、指紋など) |
| **Home** | **インテルモード**（戦術 HUD）に切り替え |
| **Insert** | **サバイバルモード**（ミニタイマー）に切り替え |
| **右クリック** | 閉じる |

## 🛡️ 安全性とコンプライアンス
本アプリケーションは**外部オーバーレイ**です。
* ✅ ゲームにコードを**注入しません**。
* ✅ ゲームのメモリを**読み取りません**。
* ✅ 安全に使用でき、アンチチートポリシーに違反しません。

## 👨‍💻 開発者向け（ポートフォリオノート）
このプロジェクトが示す技術：
* **WPF & XAML：** 複雑なUIバインディング、透明なウィンドウレイヤー、および動的なリソース辞書。
* **P/Invoke (User32.dll)：** アプリケーションのフォーカス外でのホットキー検出のためのグローバルな低レベルキーボードフックの実装。
* **MVVMアーキテクチャ：** `INotifyPropertyChanged`を使用したロジックとUIの明確な分離。
* **マルチスレッド：** 重いゲーム負荷中でもUIの応答性を確保するための高優先度ディスパッチャータイマー。
